# SimpleHTML5
SimpleHTML5

A basic template for creating HTML5 web pages with a simple navigation bar.

## Features
- Clean and minimal HTML5 structure
- Simple navigation bar for easy site navigation
- Easy to customize and extend  
- Responsive design principles
- Light and Dark mode support

## Themes

- Light Theme: Default theme with light background and dark text.
- Dark Theme: Alternative theme with dark background and light text.
- Use the settings to permanently change the theme :
    - in edge visit (copy and paste in to URL bar) edge://settings/?search=dark
    - in chrome visit (copy and paste in to URL bar) chrome://settings/?search=dark
    - in firefox visit (copy and paste in to URL bar) about:preferences#general and scroll down to "Language and Appearance" section.
